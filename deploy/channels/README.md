# Desktop channel publishing

Generated manifests live here:

- `homebrew/signet.rb`
- `winget/SignetAI.Signet*.yaml`

They are generated by:

```bash
bash deploy/channels/generate-manifests.sh \
  <version> \
  <mac_arm_url> <mac_arm_sha256> \
  <mac_x64_url> <mac_x64_sha256> \
  <win_msi_url> <win_msi_sha256>
```

## Publish flow

1. Run desktop tag build (`.github/workflows/desktop-build.yml`).
2. Download `desktop-channel-manifests` artifact.
3. Submit updates:
   - Homebrew tap: replace `Casks/signet.rb`
   - winget-pkgs: add versioned manifests under `manifests/s/SignetAI/Signet/<version>/`
4. Merge channel PRs after installer links/checksums validate.
